# Joomla WoW Recruitment Module

Allows you to set a list of classes and specs to recruit into your guild.

### Changelog

- 2015-09-22
    - fix: multiple instances of module are now possible (eg. multi raid groups)
- 2014-21-22
    - new: media folder usage
- 2014-01-19
    - new: modern layout
